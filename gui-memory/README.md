# Mixolve GUI Memory

This folder is the portable memory pack for the `mx6` GUI.

Use it when you need to:
- rebuild the same GUI language in another plugin
- keep naming, spacing, and interaction behavior consistent
- avoid re-deciding visual rules that are already settled

Files:
- `design-system.md`
- `interactions.md`
- `parameter-language.md`
- `porting-checklist.md`
- `source-map.md`
- `tokens.json`

Scope:
- visual rules
- layout rules
- control behavior
- naming rules
- transfer checklist

Non-scope:
- DSP architecture
- multiband audio routing
- host automation semantics
